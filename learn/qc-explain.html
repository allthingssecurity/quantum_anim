Intuition First

Start with three lenses and small runnable examples.

1) Superposition ~ “fair coin” until measured

// One fair coin
qc.reset(1);
qc.had(0);
qc.measure();

2) Entanglement ~ “move together”

// Lock outcomes together (00 or 11)
qc.reset(2);
qc.had(0);
qc.cnot(0,1);
qc.measure();

3) Interference ~ “small middle change, big final effect”

// H → phase → H
qc.reset(1);
qc.had(0);
qc.rz(0, Math.PI/3);
qc.had(0);
qc.measure();

Grover taste (2‑qubit toy)

qc.reset(2);
qc.had(0); qc.had(1);
qc.phase_flip('11'); // mark answer
// diffuse
qc.had(0); qc.had(1);
qc.phase_flip('00');
qc.had(0); qc.had(1);
qc.measure();

Run each example on the right with shots 200–1000 to see stable histograms.
